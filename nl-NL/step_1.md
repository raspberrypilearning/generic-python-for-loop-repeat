Om de code in een `for` lus een aantal malen uit te voeren, kun je de `range()` functie gebruiken.

```python
for item in range(10):
  print('doe iets')
```

**Tip:** Omdat deze code de waarde van `item` niet gebruikt, hoef je je geen zorgen te maken over het feit dat `range (10)` de getallen 0 tot en met 9 geeft in plaats van 1 tot en met 10 â€” er zijn nog steeds tien getallen, dus de code wordt tien keer uitgevoerd.
